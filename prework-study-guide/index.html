HTML

  • The head element contains inforation about the webpage
  • The body element represents the visible shown to the user
  • the doctype tells the browser what document type it should expect when it is open
  • The opening tag is the root of the HTML document. There are attrubutes inside the tag
  • An HTML attribute defines additional information about an element
  • Some elements used are title, style, base, link, meata
  • The title element defines webpage title
  • The style element defines the CSS styles
  • The base element defines the base URL for a webpage
  • The link element connects an external resource to the HTML document
  • The meta element defines metadata such as the character set, etc.
  • The script element embeds clide-side scripts in the HTML document

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
